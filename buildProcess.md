# Bot Build Process

Step-by-step assembly procedure for the bot.

---

## Prerequisites

- **Wall fasteners:** If any bolts or fasteners need to be pre-installed into the walls, complete that work before mounting the walls in Step 2.

---

## Assembly Steps

### 1. Start with the Bottom Plate

The bottom plate is the foundation of the assembly — everything else mounts to it.

### 2. Mount the Walls

Attach the walls to the bottom plate.

> Any wall-bound fasteners should already be installed — see Prerequisites.

### 3. Gearbox Sub-Assembly *(parallel process)*

Separately from the main assembly, mount each gearbox to its gearbox mount.

### 4. Attach Gearbox Mounts to the Bottom Plate

Once the gearbox sub-assemblies are complete, mount them onto the bottom plate.

- **At this same stage**, attach the belt and tensioner to the bottom plate.

### 5. Install the Pi Mount

After the gearbox is mounted, install the Pi mount onto the assembly.

### 6. Mount the Electronics

Install the electronics in this order:

1. Raspberry Pi
2. ESCs

### 7. Attach the Top Plate

Finish the assembly by attaching the top plate.
